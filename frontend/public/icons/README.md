# PWA Icons

To generate the required icon sizes, you can use an online tool or ImageMagick:

For now, placeholder files are created. Replace these with actual icons by:
1. Creating a 512x512 PNG icon
2. Using an online PWA icon generator, or
3. Using ImageMagick: `convert icon-512x512.png -resize 192x192 icon-192x192.png`

Required sizes: 72x72, 96x96, 128x128, 144x144, 152x152, 192x192, 384x384, 512x512
